Aprendendo sobre GIT e GitHub

Aprenda a utilizar o sistema de controle de versões Git e a plataforma GitHub! Este curso é voltado para quem deseja aprender desde os conceitos básicos até técnicas avançadas de colaboração e gerenciamento de projetos de software.

  • Introdução ao Git e Versionamento de Código
  • Uso Avançado do Git (branches, merges, rebase)
  • Colaboração em Projetos com GitHub
  • Fluxo de Trabalho em Equipe e Boas Práticas
  • Gerenciamento de Issues e Pull Requests

  • Computador com acesso à internet
  • Noções básicas de linha de comando (terminal)
  • Interesse em colaboração e versionamento de código
